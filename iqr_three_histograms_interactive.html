What students should notice

  • Dataset A (Small IQR): Values cluster tightly around the median. The middle 50% of the data fits in a narrow range. Tall, narrow histogram.
  • Dataset B (Medium IQR): Values are moderately spread. The middle 50% spans a wider range. More balanced shape.
  • Dataset C (Large IQR): Values are widely spread. The middle 50% covers a large range. Flatter, wider histogram.
Key takeaway: The IQR measures the middle 50% spread — from Q1 (25th percentile) to Q3 (75th percentile). A small IQR means values are consistent; a large IQR means values vary a lot. Formula: IQR = Q3 − Q1
